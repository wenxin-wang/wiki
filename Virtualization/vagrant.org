#+TITLE: vagrant
#+WIKI: virt

* Installation

** Debian

#+BEGIN_SRC bash
cd /tmp
wget https://releases.hashicorp.com/vagrant/2.0.0/vagrant_2.0.0_x86_64.deb
sudo dpkg -i vagrant_2.0.0_x86_64.deb
#+END_SRC

* Libvirt

#+BEGIN_SRC bash
vagrant plugin install vagrant-libvirt
#+END_SRC

** Debian

#+BEGIN_SRC bash
sudo apt-get install libvirt-dev
#+END_SRC
