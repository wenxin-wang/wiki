* 安装

** CentOS 7
#+BEGIN_SRC bash
sudo yum install -y qemu-kvm qemu-img virt-manager libvirt libvirt-python libvirt-client virt-install
sudo usermod -aG libvirt,qemu,kvm $USER 
#+END_SRC
